OPC.js

Open Platform Camera のための JavaScript ライブラリ。

OPC公式サイトへ 
"use strict";

var OPC = require("../");
var opc = new OPC();

opc.on("liveview:frame", function (jpg) {
    fs.writeFile("liveview.jpg", jpg);
});

opc.switchCameramode({
    mode: "standalone"
});

// ...

サンプルコード

Node.jsからOPCデバイスへコマンドを送信することができます。
OPC.jsに実装しているコマンドは OPC通信仕様書のコマンド一覧を参考にしており、様々な操作が可能です。
詳細は APIドキュメントを参照ください。
